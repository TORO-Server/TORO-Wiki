### 一般の方々の場合

* 荒らしに対して反応(話しかけたり、閉じ込めたり等)しないようにしてください。
  + これは[[利用規約|禁止事項]]にもあります。
  + 荒らしの行動が激しくならないようにするためです。
* 荒らされたことを必要以上に話さないでください。
  + 他の方々を不快に感じさせてしまったり、混乱させてしまったりしないようにするためです。
* 荒らされた箇所は自分で直さないで、[[Discordについて|Discord]] の [#荒らし通報](https://discord.com/channels/337838758441517057/527204678119981077) に荒らした本人の名前(MCID)とログ画像をのせてください。
  + ログ画像は [[CoreProtect]] を用いて、なるべく1〜2枚で5ブロック以上分とってください。
  + 名前によっては Discord の装飾文字に変換されてしまうことがあるので、「``」で囲むことをおすすめします。
    - 例：`Steve`
  + Discord が使えない場合は、信用できる誰かに依頼するか、[[運営チーム|運営]]に直接伝えてください。
  + 運営(Moderator 以上)がそれらを確認次第、ロールバック(復元)します。
    - 線路や額縁等の設置物は、正常に復元されないことがあります。

#### Supporter

* 荒らし本人が居る場合は、荒らしたことを確認してから「/jail 名前 牢屋名」をしてください。
* 荒らしの状況を運営チームに報告してください。

他は一般の方々の場合と同様です。

### 運営の場合

#### Moderator

* Supporterと同様です。

#### Administrator

* 荒らし本人が居る場合は、荒らしたことを確認してから「/jail 名前 牢屋名」をしてください。
* BAN をすべきかどうかを判断してください。
  + Discord に居る場合も同様です。
  + 判断が難しい場合は Owner に訊いてください。

##### 提携鯖との共有BANをする場合

1. BANレベルの際は基本的にこちらの対応でお願いします。
2. 「/minecraft:ban 名前 BAN理由」をしてください。
3. 「/sban 名前 BAN理由」をしてください。
4. 荒らされた箇所を[ロールバック](https://seesaawiki.jp/toro_server/d/CoreProtect#commandlist)してください。
5. 以上の行った事を [#荒らし通報](https://discord.com/channels/337838758441517057/527204678119981077) で報告してください。
6. 必要に応じて提携鯖の運営ユーザーに連絡を行ってください。
7. 必要に応じて「/ban-ip IPアドレス」等の処置もお願いします。

##### 共有BANをしない場合

1. 決定された処罰を行ってください。
2. 荒らされた箇所を[ロールバック](https://seesaawiki.jp/toro_server/d/CoreProtect#commandlist)してください。
3. 処罰内容・理由を「[[問題行動を起こしたプレイヤー]]」に掲載してください。
4. 以上の行った事を [#荒らし通報](https://discord.com/channels/337838758441517057/527204678119981077) で報告してください。
5. 必要に応じて「/ban-ip IPアドレス」等の処置もお願いします。

### 処罰コマンドの種類

| コマンド名 | 処罰の内容 |
| --- | --- |
| /sban ユーザ名 理由 | 提携サーバーとの共有BAN |
| /ban-ip IPアドレス 理由 | IPアドレス単位のBAN |
| /minecraft:ban ユーザー名 理由 | バニラのBAN |
| /tempban ユーザ名 数値[y/d/h/m/s] 理由 | 時間指定BAN |
| /mute ユーザ名 | チャットの禁止/禁止の解除 |
| /mute ユーザ名 数値[y/d/h/m/s] | 時間指定のチャットの禁止 |
| /jail ユーザ名 usj | 投獄 |
| /kick ユーザ名 理由 | サーバーからのキック |
| /unban ユーザ名 | EssentialsのBAN及びtempbanの解除 |
| /minecraft:pardon ユーザ名 | バニラBANの解除 |
| /unsban | sbanの解除 |
| /pardon-ip IPアドレス | ban-ipの解除 |
| /unjail ユーザ名 | 投獄の解除 |

### ヒント：荒らし対策

* [[WorldGuard]] を使って自分の作ったものを保護しましょう。
* [/tptoggle](https://seesaawiki.jp/toro_server/d/%a5%b3%a5%de%a5%f3%a5%c9%bd%b8#content_4) で荒らしに tp (テレポート)されないようにしましょう。
* 定期的に自分の作ったものを確認しましょう。
